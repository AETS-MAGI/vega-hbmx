# Experiments

このディレクトリは、テーマ単位で実験を整理するための場所です。

## 使い分け
- `daily/`: 日付単位の作業記録
- `experiments/`: テーマ単位の実験要約
- `reviews/`: 週次・節目の振り返り
- `references/`: 共通定義・命名規則・環境情報

## 命名例
- `EXP-20260307-ROCM-VS-VULKAN-QWEN/`
- `EXP-20260307-NUMGPU-SEMANTICS/`

各実験ディレクトリには、最低でも `README.md` を置き、関連する日報と run_id をリンクしてください。
