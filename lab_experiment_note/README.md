# lab_experiment_note

このディレクトリは、`vega-investigate` 配下のラボノート専用領域です。

目的は、**その日に何を試し、何を観察し、どう解釈したか**を時系列で残しつつ、必要に応じてテーマ単位の実験要約へ接続することです。

## 基本方針
- この配下には「人が読むための研究ノート」を置く
- 生ログや実測ファイルそのものは持ち込まず、関連先を参照する
- 日付単位の記録は `daily/`、テーマ単位の整理は `experiments/` に置く
- 用語、命名、共通環境は `references/` に寄せる
- 週次・節目の整理は `reviews/` に置く

## 構成
- `daily/`: 日付単位の実験記録
- `experiments/`: 実験テーマごとの要約
- `reviews/`: 週次レビュー、節目ノート
- `references/`: 用語、環境、命名ルール
- `templates/`: 新規ノート作成用テンプレート

## 運用ルール
1. 1日1ファイルを基本にする
2. その日のノートには、目的・条件・観察・解釈・次アクションを書く
3. 実験テーマが継続する場合は `experiments/` に要約を作る
4. 生データは複製せず、関連パスや `run_id` を明記する
5. 発表資料向けの文章ではなく、再現と判断根拠を優先する

## 最低限の書式
- 日報: `daily/YYYY/YYYY-MM/YYYY-MM-DD.md`
- 実験: `experiments/EXP-YYYYMMDD-NAME/README.md`
- 週報: `reviews/weekly/YYYY-Www.md`

入口は `INDEX.md` を参照してください。
